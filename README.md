# Projeto Contador Mobile

Este projeto é um aplicativo Android simples desenvolvido para fins de aprendizado de lógica de programação e manipulação de interface (UI).

## 📝 Descrição
O aplicativo consiste em um contador numérico com botões de incremento e decremento. O principal diferencial é a **lógica de cores dinâmica**: a cor do número exibido muda automaticamente a cada 10 unidades, percorrendo as 7 cores principais do espectro.

## 🛠️ Tecnologias
- **IDE:** Android Studio
- **Linguagem:** Kotlin/Java
- **Componentes:** TextView e Buttons dentro de um ConstraintLayout.
